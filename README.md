
**EMNLP 2023**

EMNLP sample.csv file contains a sample of 1000 annotated instances from the dataset. The full dataset and code will be released on acceptance.

- Column descriptions:
  - image_url: *link to the image in the social media post*
  - text: *textual content of the post*
  - cause0, cause1 and cause2: *Annotations for at most three causes for each post.*
  - class
    - 0 denotes *Non-distress*
    - 1 denotes *Distress*
